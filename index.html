Ruby & Sinatra

Numerology App

Web app created using Ruby in Sinatra framework. I've created it from scratch starting with just one .rb file which calculates birthday number and shows numerology message based on the birthdate entered by user. Then other files where added according to MVC and REST best practices. This web app can show you your numerology message and also allows you to create, edit or delete people which are saved in the database. I used Bootstrap library for style.

I've found a Skillcrush course and decided to take their Break into Tech course. I've started with basics: HTML, CSS, Git & GitHub. I've learned Ruby language, Sinatra and eventually Rails. All of this while having full-time job. It was a busy time, but I learned I’m capable of pretty much anything if I put my mind to it.
